# Rigs Monitoring

[Slush's pool](https://mining.bitcoin.cz/)

[Exchange Rates API from Blockchain](https://blockchain.info/api/exchange_rates_api)

## Installation

```console
$ git clone https://github.com/tsileo/rigsmonitoring.git
$ npm install
$ bower install
```
for development:

```console
$ grunt server 
$ grunt --force
```
